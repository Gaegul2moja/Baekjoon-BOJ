## BaekJoon/BOJ A+B_1000

단계: 입출력 받아보기
```java
import java.util.Scanner;
public class baekjoon_1000 {
    public static void main(String[] argc){
        Integer a, b;
        Scanner scanner = new Scanner(System.in);
        a = scanner.nextInt();
        b = scanner.nextInt();
        System.out.println((a+b)+"");
    }
}
```
