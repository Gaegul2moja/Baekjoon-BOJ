## BaekJoon/BOJ [JAVA] 사칙연산_10869

단계: 사칙연산 도전하기

HINT: 그냥 사칙연산이다

```java

import java.util.*;
public class baekjoon_10869 {
    public static void main(String[] argc){
        Scanner scanner = new Scanner(System.in);
        int a = scanner.nextInt();
        int b = scanner.nextInt();
        System.out.println(a+b);
        System.out.println(a-b);
        System.out.println(a*b);
        System.out.println(a/b);
        System.out.println(a%b);
    }
}

```
