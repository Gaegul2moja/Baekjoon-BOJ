## BaekJoon/BOJ [JAVA] N찍기_2741

HINT: for문을 써보자

```java

import java.util.*;
public class baekjoon_2741 {
    public static void main(String[] argc){
        Scanner scanner = new Scanner(System.in);
        int num = Integer.parseInt(scanner.next());
        for(int i=1;i<=num;i++){
            System.out.println(i);
        }

    }
}
```
